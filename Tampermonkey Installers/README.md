# Tampermonkey Installers

Static GitHub Pages copy of the Tampermonkey installer and manager UI.

- The install buttons still point to updater scripts in `ugomez809/GIA-TamperMonkey`.
- `Installer/manager.html` saves employee position assignments to this repository's `Installer/installer-config.json` copy.
- Do not place `.user.js` scripts in this repository; UGomez remains the userscript source of truth.
